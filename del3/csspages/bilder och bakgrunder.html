Bilder och background

"Den CSS backgrund-färg egenskapen anger bakgrundsfärgen för ett element."
Du kan ställa in bakgrundsfärgen för alla HTML-element.
Bakgrund kan anges med egenskapen 'background'. Background definition och använding:
  • background-color som anger en särkild färg med namne eller rag value
  • backgrund-image som anger a bild
  • background-size som anger storleken på bakgrundsbilderna
  • background-position som anger startpositionen för en bakgrundsbild
