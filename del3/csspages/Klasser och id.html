class-attribute:

Varjr HTML-element kan ha ett class -attribute. Ibland kan det bli smidigt om du identifierar ett element i ett dokument som skiljer sig från de andra elementet på sidan.
Till exemple kan du ha några stycken som innehåller information som är viktigare än andra och vill skilja mellan länkar som pekar till andra sidor på din egen webbplats och länkar som pekar på externa sidor
För att gör detta kan du använda class-attribute, värde som ett class ha bör beskriva classen den tillhör.
