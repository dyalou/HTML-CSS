CSS-sidan

Den style definiarar är sparas normalt i externa .css -filer.

Med en extern style filer kan du ändra utseendet på en hel webbplats genom att bara ändra en fil!
En CSS -regel består av en väljare och ett deklarationsblock.

Det finns tre sätt att infoga ett formatmall i CSS:

  • Extern CSS
  • Intern CSS
  • Inline CSS

External CSS

Med external style sheet kan du ändra utseendet på en hel webbplats genom att bara ändra en fil!
Varje HTML -sida måste innehålla en referens till den externa style sheet i <link> -elementet, inuti huvudsektionen.
Externa stilar definieras inom <link> -elementet, inuti den <head> element på en HTML -sida

intren CSS

Den internal style definieras i <style>element i head delen.

inline CSS

Den inline style användas för att tillämpa en unik still för ett enda element.
