Selektor i CSS

En CSS selektor välj vilken HTML- element du vill styla. Här kommer några av selektor:
  • Enkla väljare , att välj ett element baserat på namne,id,och klass.
  • Kombinationsväljare , att välj ett element baserat på ett specifikt förhållande mellan dem
