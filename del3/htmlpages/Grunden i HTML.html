Grunden i HTML

HTML är en förkortning av "HyperText Markup Language".
HTML kod börjar inom taggen <HTML>.
HTML kod består av två delar en head och en body.
<head> delen kan vi skriva alla information och instruktioer.
Den <head> elemntet är en behållare för följande element: <head>,<title> , <style>, <meta>, <link>, <script> och<base>.

  • Den HTML <head> element:

    • är en behållare för meta data och placeras mellan <html> taggen och <body> taggen. HTML-metadata är data om HTML-dokumentet metadata visas inte. Metadata definierar vanligtvis dokumenttitel,teckenuppsättning,stilar, skript och annan metainformation.

  • Den HTML <body> element:

    • definierar dokumentets kropp, och innehåller rubriker, stycken, bilder, tabeller, listor etc.

  • Den HTML <title> element:

    • Elmentet definierar dokumentets titel och krävs i HTML dokument.
    Elmentet <title>:
    1. definierar en titel i webbläsarens verktygsfält.
    2. ger en titel för sidan när den läggs till i favoriter
    3. visar en titel för sidan i sökmotorresultat

  • Den HTML <style> element:

    • Elementet <style> används för att definiera stilinformation för en enda HTML -sida:

  • Den HTML <link> Element:

    • Elementet <link> definierar förhållandet mellan det aktuella dokumentet och en extern resurs.

  • Den HTML <meta> Element:

    • En <meta> tag som definierar meta data om ett HTML-dokument. Denna tag sitter i <head> element och kommer inte att visas på sidan. Meta data användas av webbläsare (hur man visar innehållet eller laddar om sidan),sökmotor och andra webbtjänster.
    Elementet <meta> används vanligtvis för att ange teckenuppsättning, sidbeskrivning, nyckelord, dokumentförfattare och visningsportinställningar. <body> tag definierar dokumentets kropp och är en behållare för allt synligt innehåll, till exempel rubriker, stycken,bilder, listor etc.

    HTML block och inline elements

    Varje HTML -element har ett visa värde, beroende på vilken typ av element det är. Det finns två visa värden: block och inline.

    Block-level elements

    Denna element börjar alltid på en ny rad. Ett block-level element tar alltid upp hela den tillgägliga bredden. Ett block element har en över och en nedre margin, medan ett inline element inte har det. Den element är en block-level element.
    DIV element

    Den <div> elementet användas ofta som en behållare för andra HTML-element. Elementet <div> har ingen obligatoriska attribute, men style, class och id är vanliga. När detta elementet användas tillsammans med CSS Kan elementet <div> användas för style innehållet av block.
    Det finns många block-level element i HTML här kommer några av dem:
    <footer> , <table> , <header> , <form> , <section> , <article> , <h1>-<h6> .

    Inline element

    Denna element startar inte på en ny rad. Ett inline element tar bara upp så mycket bredd som behövs. Här kommer några av inline element: <a> , <img> , <em> , <script> , <strong>.
    Den <span> elementet är ett inline-element och starta inte på en ny rad och tar bara upp så mycket bredd som behövs. Example på inline element med span element:

    This is four themes since, architecture, history and religion which is necessary for learning in the schools.
